# League of Legends Data Pipeline (Riot API)

Этот проект реализует ETL-процесс (Extract, Transform, Load) для сбора данных о топ-игроках **Challenger-лиги EUW**, их матчах и статической информации о чемпионах. Результатом являются три структурированных CSV-файла, готовых для дальнейшего анализа (расчёт винрейтов, KDA, популярности чемпионов, корреляций и т.д.). Также создаётся интерактивный дашборд на основе Plotly.

---

## Установка и настройка

### 1. Клонируйте репозиторий (или загрузите файлы)

```bash
git clone https://github.com/yourusername/lol-data-pipeline.git
cd lol-data-pipeline
```

### 2. Получите API-ключ Riot Games

1. Зарегистрируйтесь на [Riot Developer Portal](https://developer.riotgames.com/).
2. Создайте приложение и получите **Production API Key** (или Development – для тестов).
3. В корневой папке проекта создайте файл `riot_key.env` и запишите в него:

```env
RIOT_API_KEY=ваш_api_ключ
```

## Описание проекта

1. **Extract**  
   - Получение списка топ-310 игроков Challenger-лиги EUW.  
   - Для первых 5 игроков загружено по 5 последних матчей (итого 250 записей участников).  
   - Загрузка справочника 167 чемпионов с Data Dragon.

2. **Transform**  
   - Извлечение нужных полей из JSON (PUUID, статистика, теги чемпионов).  
   - Расчёт KDA: `(kills + assists) / max(deaths, 1)`.  
   - Объединение таблиц матчей, чемпионов и игроков в единый `final_df`.

3. **Load**  
   - Сохранение трёх CSV: `players.csv`, `matches.csv`, `champions.csv`.  
   - Сохранение объединённой таблицы `final_data.csv`.  
   - Генерация интерактивного HTML-дашборда (`dashboard.html`) с 5 графиками.

## Интерактивный дашборд

Дашборд (`dashboard.html`) содержит 5 графиков:

1. **Топ-10 чемпионов по винрейту** – столбчатая диаграмма с градиентом.
2. **Топ-10 чемпионов по суммарному LP** – столбчатая диаграмма.
3. **Распределение LP игроков** – гистограмма.
4. **Длительность матчей** – диаграмма размаха (boxplot) с точками.
5. **Средние убийства vs смерти по чемпионам** – scatter-plot с размером точек (число матчей) и линией тренда.

Дашборд открывается в браузере и полностью интерактивен (наведение, масштабирование).

## Важные замечания
- **Обработка ошибок**: код выводит статусы ответов. Если запрос вернул 403 или 429 – проверьте ключ или снизьте частоту запросов.
- **Регион**: проект ориентирован на EUW. Для других регионов измените переменные `REGION` и `MATCH_REGION` в коде.
